The simplest example I can think of requires at least two scripts

  1. An entry-point script (main.js)
  2. Some simple dependency script (dep.js)

And, of course, a page to load all of this, simple.html

And to keep the example super-simple, these scripts will live alongside require.js so we have a structure like:

    simple.html
    `- js
        |- require.js
        |- main.js
        `- dep.js
simple.html forms the entry-point for the application. To get RequireJS working, this is the place where we will reference require.js. We also add a data-main attribute to the script inclusion with a path relative to the location of index.html for RequireJS to load up. We'll use main.js for this. So our simple.html could look like:
